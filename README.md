# EXPO 2026 – IAU Graduation Projects Platform

## Overview
EXPO 2026 is a web-based platform developed for Imam Abdulrahman Bin Faisal University (IAU) to manage and showcase graduation projects.  
The system allows students to upload their projects, judges to evaluate submissions, and administrators to manage the entire exhibition process.

---

## Features

### Student Features
- Student registration and login
- Upload graduation projects and posters
- View project status and evaluation results
- Receive certificates after evaluation

### Judge Features
- Judge login system
- Evaluate projects using scoring forms
- Submit comments and feedback
- Generate final project results

### Admin Features
- Manage students and judges
- Control project submissions
- View all evaluations
- Generate certificates automatically
- Manage homepage content and announcements

### General Features
- Responsive modern UI
- Arabic / English translation support
- Search functionality
- Dynamic project display
- Database integration using MySQL
- Email support using PHPMailer
